# 📚 Calc-Avg Official Documentation Hub

Welcome to the official documentation directory for the **Calc-Avg** project! This wiki contains all the technical details, release histories, compilation instructions, and community guidelines for the software.

## 📂 Directory Structure

Please navigate through the folders below to find the specific information you are looking for:

- **[features/](features/)**: Detailed explanations of all the advanced mathematical engines, parsers, and UI components available in the software.
- **[how_to_run/](how_to_run/)**: Step-by-step guides on how to execute the pre-compiled binaries or run the project from source.
- **[make_executable_files/](make_executable_files/)**: Compilation instructions (`gcc` and `Makefile`) to build the project yourself on Windows, Linux, and macOS.
- **[requirements/](requirements/)**: The system dependencies and prerequisites required to run the C networking and statistical engines.
- **[system_supported/](system_supported/)**: A comprehensive list of all Operating Systems natively supported by Calc-Avg.
- **[releases/](releases/)**: The complete version history of the software. From the `1.0` foundation, all the way to `2.2` (The Networking Update).
- **[users_suggest/](users_suggest/)**: The official community ledger. Use the templates provided here to suggest new features or submit opinions to the creator.
- **[Follow-Sanskar/](Follow-Sanskar/)**: Links to follow the creator of this open-source project, including support emails and social media platforms!
