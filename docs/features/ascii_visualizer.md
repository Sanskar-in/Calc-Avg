# ASCII Data Visualizer

Calc-Avg doesn't just crunch numbers; it visualizes them.

## The Visual Engine
Before calculating the mathematical average of a custom array, Calc-Avg triggers the `print_ascii_bar_chart()` algorithm.

**Process:**
1. It scans your entire dataset to find the absolute maximum value.
2. It sets this maximum value as the 100% scale marker.
3. For every number you entered, it calculates its exact percentage relative to the maximum value.
4. It dynamically draws a vivid Green ASCII Bar Chart in the terminal, scaled perfectly to 40 characters wide.

This instantly provides you with a comprehensive visual understanding of how your data is distributed, spotting massive outliers visually before looking at the numerical average!
