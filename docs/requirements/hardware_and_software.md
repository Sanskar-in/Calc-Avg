# System Requirements

Calc-Avg is incredibly lightweight and optimized.

## Hardware Requirements
- **CPU:** Any modern processor (x86, x64, ARM).
- **RAM:** Less than 5 MB of RAM required for execution.
- **Storage:** Less than 1 MB of disk space.

## Software Requirements
**For End Users:**
- A terminal emulator (Command Prompt, PowerShell, GNOME Terminal, iTerm2).
- The compiled executable file (`calc_avg.exe` or `calc_avg`).

**For Developers (To compile from Source):**
- A C compiler that supports standard C99 or higher (GCC, Clang, MSVC).
- Make (Optional, to use the provided Makefile on Unix systems).
