# How to Run: Interactive Mode

The Interactive Mode provides a beautiful, ANSI-colored terminal UI with a 20-option menu.

## Steps to Run

1. Open your terminal (Command Prompt, PowerShell, Terminal, bash).
2. Navigate to the project directory containing the `.exe` (or Linux binary).
3. Execute the file directly:
   - **Windows:** `.\calc_avg.exe`
   - **Linux/macOS:** `./calc_avg`

## Navigating the UI

- Upon launching, you will be greeted with the official Copyright Banner.
- A Cyan and Yellow menu will appear asking for your choice (1-20).
- Type the number of the mathematical operation you wish to perform and press Enter.
- Follow the on-screen prompts (e.g., entering `N` or typing out your numbers one by one).
- The result will be printed in bold Green, and automatically saved to `history.log` in the exact same folder!
