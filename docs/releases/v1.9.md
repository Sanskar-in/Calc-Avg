# Release Notes: Version 1.9

## Overview
Version 1.9 is the "Visual Data" update, proving that the terminal is not limited to just text and numbers. 

## Key Features Implemented
- **ASCII Data Visualizer:** Built the highly complex `print_ascii_bar_chart()` algorithm.
- **Functionality:** Before any array is averaged (either interactively or via the CLI), the visualizer scans the dataset, finds the absolute maximum value, scales it to 100%, and uses Green ASCII blocks (`#`) to draw a perfect proportional bar chart for every single number. This allows users to instantly spot massive outliers in their data distribution before reading the final mean.
