# Release Notes: Version 2.2 (THE NETWORKING UPDATE)

## Overview
Calc-Avg officially steps into the realm of modern internet architecture! Version 2.2 introduces the ability to connect to external internet servers via raw TCP sockets. The C program can now download live JSON data via HTTP GET requests and instantly calculate averages in real-time.

## Key Features Implemented
- **Raw C Socket Networking:** Integrated Windows Sockets 2 (`winsock2.h`). The program natively initializes networking interfaces, resolves hostnames to IPv4 addresses, and creates TCP streams on Port 80.
- **HTTP Request Parsing:** Built the `calc_avg_from_api()` engine. It sends raw HTTP/1.1 `GET` requests to any provided hostname and path.
- **Live Data Bridging:** Once the API responds, the C engine intercepts the byte stream, intelligently strips away all HTTP response headers, extracts the pure JSON payload, saves it to memory, and pipes it directly into the Version 2.0 `calc_avg_from_json()` state machine to extract the numerical data!
- **Interface Integration:** Accessible via the interactive menu (Option 22) or instantly through the CLI using the new `--api` flag (e.g. `calc_avg.exe --api jsonplaceholder.typicode.com /users/1`).

## Binary Archive
