# Release Notes: Version 3.0 (THE WEB SERVER & SONIFICATION UPGRADE)

## Overview
Welcome to the legendary **Version 3.0**. The Calc-Avg project has officially transcended the boundaries of a standard desktop terminal application. We have engineered a Native HTTP Web Server and a Hardware Audio Sonification Engine directly into the C codebase!

## Key Features Implemented

### 1. The Web Server Update (Browser-Based Frontend)
- **Native HTTP TCP Socket Server:** We engineered `src/server.c`, utilizing `winsock2.h` to open Port 8080. Calc-Avg now actively hosts a live website on your local network!
- **Beautiful HTML/CSS/JS Interface:** You can open Google Chrome or Microsoft Edge, type `http://localhost:8080`, and interact with a premium styled Web Application.
- **Dynamic API Routing:** The C engine parses the URL parameters (`/calc?data=x,y,z`), mathematically calculates the average, and dynamically injects the result back into the HTML response!

### 2. Data Sonification (Hardware Audio Generation)
- **Physical Sound Synthesis:** We integrated the low-level Windows kernel API `Beep()` inside `src/audio.c`.
- **Musical Averages:** Calc-Avg will mathematically map your massive data arrays into physical audio frequencies (Hertz) and play them as a musical sequence through your computer speakers, culminating in a grand synthesized note for the final calculated average!
