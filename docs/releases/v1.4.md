# Release Notes: Version 1.4

## Overview
Version 1.4 focused on User Experience (UX) and data filtering. This patch transformed the application from a standard monochrome terminal tool into a premium visual experience.

## Key Features Implemented
- **Premium ANSI UI:** Completely overhauled the `printf` engine using standard ANSI escape codes. 
  - Menus are wrapped in Cyan borders.
  - Options are highlighted in Yellow.
  - Success outputs are bold Green.
  - Errors are highlighted in Red.
- **Filtered/Range-Based Averages:** Implemented `calc_filtered_average()`. Users can input massive datasets and set strict Minimum/Maximum boundaries. The engine automatically drops anomalies and outliers before calculating the final mean.
