# Release Notes - Version 5.2.0.0 (The AI POSIX Patch)

## Overview
Version 5.2 serves as a critical hotfix to address a fatal architectural bug discovered in the Deep Learning AI engine (`neural_net.c`). 

While Version 5.1 hardened the HTTP server threads, it was discovered that the `start_neural_network_training` function was completely skipping Thread creation on Linux environments due to a missing POSIX `#else` block around the raw `_beginthread` Windows API. This effectively bricked the AI Web Dashboard on UNIX platforms.

## Key Features & Changes
- **Deep Learning Threading Cross-Platform Fix:** Injected the missing POSIX `pthread_create` fallback into the Neural Network startup routine.
- **Thread Signature Refactor:** Updated `train_neural_network_thread` to return a standard UNIX `void*` instead of `void`, preventing severe compilation warnings on strict POSIX build environments.
- **Sleep Abstraction:** Safely wrapped the internal epoch delay loops (used to yield CPU cycles to the WebSocket broadcasting thread) with native `usleep()` for Linux.

*Calc-Avg's Deep Learning AI now trains flawlessly on UNIX systems. Signed off by Sanskar - 2026*
