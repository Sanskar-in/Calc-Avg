# Release Notes - Version 5.5.0.0 (The Native GUI Polish Update)

## Overview
Calc-Avg's Native Win32 Graphical User Interface (`gui.c`) has historically remained a neglected artifact of Version 2.8. In Version 5.5, we have formally synchronized the Win32 canvas to reflect the final Masterpiece state of the repository.

## Key Features & Changes
- **Modern Typography Injection:** Injected a `WM_SETFONT` message loop into the `WM_CREATE` routine. The GUI now overrides the ugly default Windows 95-style font, dynamically loading and applying `Segoe UI` to all child components (Text Inputs, Buttons, and Labels) for a clean, modern aesthetic.
- **Window Title Synchronization:** The `CreateWindowW` Win32 API call has been updated. The Native GUI window now proudly displays `Calc-Avg V5.5 Graphical User Interface` instead of the outdated `2.8` string.

*Calc-Avg's Native Win32 UI now accurately reflects its underlying Masterpiece C engine! Signed off by Sanskar - 2026*
