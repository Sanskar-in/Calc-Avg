# Release Notes: Version 4.5 (THE CROSS-PLATFORM EXPANSION)

Calc-Avg has achieved true cross-platform capabilities! **Version 4.5** introduces the massive **POSIX Cross-Platform Expansion**. We have carefully refactored the entire C Engine to abstract away hardcoded Windows APIs (`winsock2.h`, `windows.h`). 

The high-performance API Microservice now natively utilizes `pthread.h` and `<sys/socket.h>` to compile seamlessly on Linux and macOS environments. The custom JSON parsing state machine has also been completely overhauled to explicitly map key-value pairs rather than indiscriminately scraping numbers. 

The compiled executable for Version 4.5 (equipped with the complete POSIX fallbacks, robust JSON parsing, Neural Networking, Surveillance Engine, and Version 4.5.0.0 Metadata) is entirely archived in:
`versions/4.5/calc_avg.exe`
