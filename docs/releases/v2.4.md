# Release Notes: Version 2.4 (THE SQLITE INTEGRATION)

## Overview
Calc-Avg officially steps into native Database Analytics! Version 2.4 introduces the ability to connect to and parse **SQLite Databases** natively. Without needing any external SQL servers, the C engine now directly opens `.sqlite` or `.db` files, dynamically executes SQL Queries, extracts numerical data, and computes real-time averages.

## Key Features Implemented
- **Native SQLite C Engine:** We downloaded and integrated the official SQLite C Amalgamation (`sqlite3.c` and `sqlite3.h`) into the `third_party/sqlite/` directory.
- **SQL Query Generation:** Implemented `calc_avg_from_sqlite()`. It utilizes `sqlite3_open`, constructs a dynamic `SELECT <column> FROM <table>` query, executes it with `sqlite3_prepare_v2`, and extracts the pure float values via `sqlite3_step`.
- **Interface Integration:** Accessible via the interactive menu (Option 24) or instantly through the CLI using the new `--sqlite` flag (e.g. `calc_avg.exe --sqlite test.db financials revenue`).
