# Release Notes: Version 3.6 (THE NEURAL NETWORK EXPANSION)

## Overview
Calc-Avg is now a true Deep Learning Engine! **Version 3.6** completely expands the AI capabilities of the native C backend by integrating a from-scratch Multi-Layer Perceptron (Neural Network). The engine trains the AI model asynchronously and dynamically injects the loss function into the WebSocket stream, allowing the Web Dashboard to visualize the AI getting smarter in real-time!

## Key Features Implemented

### 1. Natively Engineered Deep Learning (`src/neural_net.c`)
- **Multi-Layer Perceptron (MLP)**: Built a 1-Hidden-Layer neural network architecture natively in C without external AI libraries.
- **Advanced Mathematics**: Implemented Forward Propagation (Dot products and Sigmoid Activation) and Backpropagation (Gradient Descent and Chain Rule derivatives) natively.
- **Asynchronous Execution (`_beginthread`)**: When the user requests AI training, the C engine dynamically spawns a background thread. This allows the Neural Network to perform 10,000 epochs of intense mathematical computation while keeping the main HTTP/WebSocket server lightning fast!

### 2. WebSocket Real-Time Telemetry Upgrades (`server.c`)
- **Dynamic JSON Injection**: Upgraded the `websocket_stream_thread()` logic. While the background neural network thread computes, the WebSocket thread safely monitors global variables (`global_nn_epoch`, `global_nn_loss`) and seamlessly injects them into the 5Hz WebSocket JSON blast alongside the Remote Desktop screen buffer!
- **SQLite History Logging**: The final AI predictions and loss values are securely logged into the SQLite Database via `log_to_db`!

### 3. Deep Learning Web Dashboard (`web-app/`)
- **New Deep Learning AI Tab**: Added an ultra-modern Glassmorphism tab for submitting array sequences for Neural Network predictive training.
- **Animated Chart.js Loss Graph**: Programmed `script.js` to parse `data.nn_loss` from the WebSocket payload, dynamically plotting the Neural Network's Cost Function on a beautiful line chart. You can literally watch the AI learn and minimize error in real-time!
