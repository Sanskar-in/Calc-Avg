# Release Notes: Version 3.4 (THE BRAND EXPANSION)

## Overview
Calc-Avg has officially established its visual identity! **Version 3.4** introduces the **Brand Expansion**, integrating a custom, AI-designed futuristic glassmorphism logo into both the Web Dashboard and the native Windows executable binary!

## Key Features Implemented

### 1. The Calc-Avg Logo
- **Futuristic Aesthetic**: A custom, high-tech neon blue and purple logo featuring an integrated mathematical average/integral symbol, designed with a sleek glassmorphism vibe perfectly matching the Web Dashboard's aesthetic.

### 2. Native Windows Iconography (`src/calc_avg.rc`)
- **.ICO Embedding**: The raw `.png` logo was converted into a multi-resolution `.ico` icon file and natively embedded directly into the `.exe` via the Windows Resource Compiler.
- **Windows File Explorer**: The `calc_avg.exe` file no longer shows a generic blank application icon! It proudly displays the high-tech glowing Calc-Avg logo natively in the Windows Desktop and File Explorer!

### 3. Web Dashboard Branding (`web-app/`)
- **Favicon Integration**: The HTML `<head>` now pulls the logo as a `favicon.ico`, replacing the blank globe icon in the Chrome browser tab with the glowing neon logo.
- **C Web Server Routing**: Upgraded the `server.c` HTTP routing logic to properly intercept and serve `image/png` MIME types for the logo file, feeding it directly to the browser.
- **App Header Overhaul**: The Web Dashboard's main header was redesigned utilizing Flexbox and custom CSS to prominently display the glowing logo alongside the "Calc-Avg v3.4" title, creating a commercial-grade, premium feel!

## Binary Archive
The compiled Windows executable for Version 3.4 (equipped with Native WebSockets, CPU/RAM APIs, GDI Screen Capture, the Embedded Application Icon, and Version 3.4.0.0 Metadata) is completely archived in:
`versions/3.4/calc_avg.exe`
