# Release Notes: Version 3.7 (THE VOICE SURVEILLANCE EXPANSION)

## Overview
Calc-Avg has officially evolved into a full multimedia streaming platform! **Version 3.7** introduces a massive **Voice Surveillance** feature. The C Server now directly taps into the host computer's microphone hardware, encodes the raw audio waves in real-time, and streams them simultaneously with the remote desktop screen over the high-speed 64-bit WebSocket. You can now physically watch and listen to the remote machine simultaneously directly from your browser!

## Key Features Implemented

### 1. Hardware Microphone Access (`src/audio.c`)
- **Windows Multimedia Kernel APIs**: Programmed low-level hardware access using `<mmsystem.h>` and `waveInOpen` to physically capture raw sound waves from the machine's default microphone.
- **Real-Time PCM Recording**: The C backend continuously records high-quality 8000Hz 8-Bit Mono audio into a ring buffer of 1600-byte packets, perfectly synchronized to match the 5Hz screen streaming frame rate!
- **On-the-Fly Cryptography (`CryptBinaryToStringA`)**: The raw audio buffers are instantly encoded into a compressed Base64 string directly in the microphone callback thread (`waveInProc`) using native Windows Cryptography APIs.

### 2. Upgraded JSON WebSocket Streaming (`src/server.c`)
- **Seamless JSON Injection**: The `websocket_stream_thread()` logic was heavily upgraded. Every 200 milliseconds, it now retrieves the latest microphone buffer via `get_latest_mic_base64()`, alongside the screen buffer, CPU, and RAM data.
- **Combined A/V Stream**: The server builds a massive JSON packet containing both Video (`data.screen`) and Audio (`data.mic`) and blasts it over the active WebSocket simultaneously!

### 3. Voice Surveillance Dashboard (`web-app/`)
- **Web Audio API Decoding**: Upgraded `script.js` to intercept the massive incoming WebSocket packets. When the `"mic"` Base64 data arrives, it is dynamically decoded, converted back into a `Float32Array` of PCM data, and pumped directly into the browser's `AudioContext` to perfectly recreate the microphone sound on your speakers in real-time!
- **Live Canvas Audio Visualizer**: Programmed a beautiful HTML `<canvas>` animation that maps the raw audio float array into a glowing purple waveform! As the remote user speaks into the microphone, the Web Dashboard physically draws their soundwaves on the screen!
