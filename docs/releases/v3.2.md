# Release Notes: Version 3.2 (THE PREMIUM METADATA UPDATE)

## Overview
Calc-Avg has officially reached commercial-grade software status! **Version 3.2** introduces native Windows Executable Metadata integration, ensuring the software presents a highly professional, premium finish to all end-users directly within the operating system GUI.

## Key Features Implemented

### 1. Windows Portable Executable (PE) Metadata Injection
- **Native Resource Compiler (`calc_avg.rc`):** Engineered a native C-Resource script containing the official Windows `VERSIONINFO` block.
- **Embedded Properties:** When an end-user hovers their mouse over `calc_avg.exe` in Windows File Explorer (or right-clicks and navigates to Properties > Details), they will natively see:
  - **Company Name:** `Sanskar`
  - **File Description:** `Advanced tool to calculate the average of any numbers via C engine`
  - **File/Product Version:** `3.2.0.0`
  - **Legal Copyright:** `Copyright (c) Sanskar. All rights reserved.`
  - **Product Name:** `Calc-Avg Framework`

### 2. Build System Architectural Upgrade
- **`windres` Integration:** The project `Makefile` has been overhauled to officially integrate the `windres` Resource Compiler into the GNU/MinGW build chain. The `.rc` script is compiled into an unlinked `.res` object and seamlessly bound into the final `.exe` binary during the GCC compilation phase.

## Binary Archive
The compiled Windows executable for Version 3.2, complete with the embedded premium metadata properties, is safely stored in:
`versions/3.2/calc_avg.exe`
