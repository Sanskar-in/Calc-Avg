# Release Notes: Version 2.6 (THE FINANCIAL MARKET UPDATE)

## Overview
Calc-Avg steps onto Wall Street! Version 2.6 implements quantitative trading algorithms to allow for the mathematical analysis of financial markets, volatile stock assets, and long-term trend momentum. 

## Key Features Implemented
- **Simple Moving Average (SMA):** Input an array of historical stock prices and a custom Window Period (e.g. 5, 14, 50, or 200 Days). The C engine implements a sliding-window array to compute the pure unweighted arithmetic mean over that rolling period.
- **Exponential Moving Average (EMA):** Unlike the SMA which weights every day equally, the new EMA algorithm dynamically calculates an exponential multiplier (`2 / (Window + 1)`) to apply exponentially heavier mathematical significance to recent price action, tracking market momentum faster.
- **Financial Table Renderer:** Both algorithms execute concurrently, mapping your input into a beautifully structured ASCII trading table (`Day | Price | SMA | EMA`) to easily compare lagging and leading trend indicators.
