# Release Notes: Version 2.8 (THE ULTIMATE GRAPHICS UPDATE)

## Overview
We have conquered the terminal, and now we conquer the Desktop. Version 2.8 breaks Calc-Avg out of the command line and spawns a **True Native Graphical User Interface (GUI)** using the Windows Win32 API. 

## Key Features Implemented
- **Native OS Desktop Window:** Bypassing all text-based limitations, the C engine now invokes `<windows.h>` and `-lgdi32` to compile an authentic desktop application window.
- **Physical Input Controls:** Instead of standard input/output streams (`printf`/`scanf`), users can now type their numbers into an actual physical `EDIT` Textbox widget.
- **Mouse Event Hooks (WndProc):** The engine establishes a Message Event Loop and Window Procedure to listen for physical hardware interrupts (Mouse Clicks). Clicking the standard `BUTTON` widget triggers the calculation.
- **Visual Rendering:** The math results are dynamically drawn onto the GUI canvas using a `STATIC` text label instead of being printed to a terminal.
- **Unified Engine:** The GUI operates entirely in tandem with the command-line CLI. Option 28 in the terminal will seamlessly spawn the independent visual Desktop Application.
