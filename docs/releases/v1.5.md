# Release Notes: Version 1.5

## Overview
Version 1.5 is a branding update, heavily establishing the project's identity as an open-source tool built by the original creator.

## Key Features Implemented
- **Copyright Terminal Banner:** Built a function inside `utils.c` that renders a massive, bold golden banner at the very top of the application upon boot. It explicitly attributes the copyright to **Sanskar** and provides the GitHub repository link to all users before they even enter the interactive menu.
- **Source File Headers:** Every single `.c` and `.h` file in the source code was updated to include the official copyright header for legal open-source distribution.
