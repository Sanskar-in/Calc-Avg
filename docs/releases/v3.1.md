# Release Notes: Version 3.1 (THE WEB REST API & DASHBOARD UPGRADE)

## Overview
Calc-Avg just received a massive architectural overhaul for its web capabilities. **Version 3.1** transforms the backend server into a true decoupled JSON REST API microservice and introduces a stunning, premium Web Dashboard powered by modern JavaScript and Chart.js.

## Key Features Implemented

### 1. File System Web Server (`src/server.c`)
- **Static Asset Serving:** The TCP HTTP Server now actively reads physical files from the `web/` directory instead of returning hardcoded strings. It seamlessly serves `.html`, `.css`, and `.js` resources to the browser.

### 2. JSON REST API Microservice
- **`/api/calculate` Endpoint:** The C backend intercepts API requests, computes comprehensive statistical mathematics (Mean, Median, Mode, Variance, Standard Deviation), and generates structured JSON string responses. 

### 3. Modern Web Dashboard Frontend (`web/index.html`)
- **Glassmorphism Aesthetics:** The interface has been completely redesigned using modern CSS, featuring dynamic animated orbs, glass panels, beautiful hover micro-animations, and premium Google Fonts (`Inter` & `Outfit`).
- **Dynamic Fetch API:** The frontend JavaScript dynamically hits the C backend REST API using asynchronous `fetch()`, updating the UI instantly without reloading the webpage.

### 4. Interactive Data Visualization
- **Chart.js Integration:** The web dashboard utilizes `Chart.js` via CDN. When mathematical results are returned from the C server, the JavaScript engine seamlessly renders an animated Bar Chart representing the raw data arrays natively inside the browser!
