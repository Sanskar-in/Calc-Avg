# Release Notes: Version 2.5 (THE MATRIX MATHEMATICS UPDATE)

## Overview
Calc-Avg moves beyond 1-dimensional lists! Version 2.5 introduces a sophisticated algorithm to process and analyze **2D Matrices**. You can now input an entire grid of data (like a spreadsheet or image pixel matrix), and the C engine will simultaneously map and calculate the averages across every dimension.

## Key Features Implemented
- **Row-by-Row Analytics:** The engine iterates across the X-axis of your matrix, outputting the mathematical average of every single row.
- **Column-by-Column Analytics:** Simultaneously, it maps the Y-axis, yielding the average of every individual column.
- **Total Matrix Convergence:** After isolating the X and Y bounds, it aggregates the entire 2D field into one master mathematically precise total average.
- **ASCII Grid Visualizer:** The UI dynamically formats the raw numbers into a beautiful, ANSI-colored 2D grid right inside the terminal, graphing row averages on the right edge (in Yellow) and column averages on the bottom edge (in Green).
