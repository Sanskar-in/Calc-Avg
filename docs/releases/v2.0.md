# Release Notes: Version 2.0 (THE SPREADSHEET UPDATE)

## Overview
Welcome to Version 2.0! This release marks a massive architectural leap for Calc-Avg. It transitions the software from a standard math calculator into a legitimate data science utility capable of processing structured enterprise data files in raw C.

## Key Features Implemented
- **CSV Spreadsheet Parsing:** Implemented the `calc_avg_from_csv()` engine. Users can now pass massive Excel `.csv` files into the calculator. The custom C parser utilizes `strtok` to strip commas and seamlessly aggregates all numerical spreadsheet data instantly.
- **JSON Data Parsing:** Implemented the `calc_avg_from_json()` engine. Built a custom character-by-character state machine to parse JavaScript Object Notation files. It intelligently ignores structural braces `[]{}`, quotes `""`, colons `:`, and text strings, seeking out only pure floating-point numbers to calculate their average.
- **CLI & Interface Integration:** Both of these massive engines are fully accessible through the interactive ANSI menu (Options 19 and 20) and via hyper-fast command-line arguments (`--csv` and `--json`).
