# Release Notes: Version 2.3 (THE MULTI-THREADING UPDATE)

## Overview
Calc-Avg officially leaps into Enterprise-Grade Performance! Version 2.3 completely rewrites the batch processing architecture to support **True Multi-Threading / Parallel Processing** utilizing native Win32 Threads!

## Key Features Implemented
- **Parallel Data Processing:** When you pass multiple files into Calc-Avg (via Option 18 or `--batch`), the engine no longer waits for one file to finish before starting the next.
- **Windows Threads (`<windows.h>`):** The application now natively interfaces with the Windows operating system using `CreateThread` and `WaitForMultipleObjects`.
- **CPU Optimization:** It literally spins up an independent background thread for *every single file* provided. These threads run entirely in parallel across your processor's cores, instantly slashing processing time for massive spreadsheet operations.
- **Dynamic Thread Aggregation:** Once all threads finish simultaneously, the main application mathematically aggregates all local sums and local counts, merging them back into one master mathematically precise average!
- **Cross-Platform Graceful Fallback:** If you compile this code on Linux or macOS (where `<windows.h>` does not exist), the C Preprocessor automatically detects the OS and gracefully falls back to the classic, highly-stable single-threaded batch processor, ensuring 100% universal compatibility!
