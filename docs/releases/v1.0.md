# Release Notes: Version 1.0

## Overview
Version 1.0 marks the initial foundation of the **Calc-Avg** project. The core focus of this release was to establish the architectural skeleton of the application, proving the concept of a terminal-based calculator capable of handling dynamic user input without crashing.

## Key Features Implemented
- **Infinite Menu Loop:** Established the primary `while(1)` loop in `main.c` to keep the application running until the user explicitly chooses to exit.
- **Dynamic Memory Allocation:** Used `malloc` to allow users to input an arbitrary number of floating-point values on the fly without rigid array bounds.
- **Basic Mean Engine:** Implemented the core `calc_avg_array()` function to calculate the standard arithmetic mean of the user's dataset.
