# Release Notes: Version 1.3

## Overview
Version 1.3 is the "Data Persistence & Statistics" update. The calculator was upgraded into a true statistical tool, and a background logging engine was created to track user behavior.

## Key Features Implemented
- **Advanced Statistical Models:** 
  - Integrated `calc_median()` and `calc_mode()` leveraging the built-in `qsort` algorithm.
  - Integrated `calc_geometric_mean()` and `calc_harmonic_mean()` with strict zero/negative value safeguards.
- **Weighted Averages:** Users can now assign specific weights to data points (e.g. for calculating GPAs).
- **History Logging Engine:** Built the `log_history()` function. The application now silently appends every calculation (including timestamps via `<time.h>`) directly to a `history.log` file.
- **Data Exporting:** Added the ability to export specific active results to a clean `results.txt` file.
