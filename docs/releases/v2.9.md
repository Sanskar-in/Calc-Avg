# Release Notes: Version 2.9 (THE ARTIFICIAL INTELLIGENCE UPDATE)

## Overview
Calc-Avg can now predict the future. Version 2.9 integrates a powerful Predictive Artificial Intelligence engine written natively in pure C. The engine uses advanced linear algebraic and statistical calculations to train a Machine Learning model directly from your historical data arrays.

## Key Features Implemented
- **Predictive Linear Regression:** The C engine iterates across your array of data, mapping `x` and `y` coordinates, and extracting the perfect mathematical sums of `x`, `y`, `x^2`, and `xy`.
- **Model Training:** Using the advanced Regression formula, the engine calculates the exact Slope (m) and Y-Intercept (b), formulating a perfectly optimized `y = mx + b` Line of Best Fit.
- **Future Generations:** Once the model is trained, you can command the engine to predict exactly what the data values will be in future, nonexistent time periods. The engine dynamically calculates and generates AI predictions.
- **AI Table Renderer:** The engine formats your historical data alongside its own predictive generated values inside a flawlessly mapped, ANSI-colored terminal table.
