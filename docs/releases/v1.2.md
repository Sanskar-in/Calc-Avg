# Release Notes: Version 1.2

## Overview
Version 1.2 focused on Community Engagement. As an open-source tool, it became necessary to build a bridge directly between the terminal application and the online repository.

## Key Features Implemented
- **GitHub Redirection Engine:** Added a new option to the interactive menu allowing users to instantly open the project's source code.
- **Cross-Platform System Calls:** Leveraged C preprocessor directives (`#if defined(_WIN32)`) to dynamically trigger the correct operating system command for opening URLs:
  - Windows: `system("start ...")`
  - macOS: `system("open ...")`
  - Linux: `system("xdg-open ...")`

## Binary Archive
The compiled Windows executable for this specific version is safely stored in:
`versions/1.2/calc_avg.exe`
