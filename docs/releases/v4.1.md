# Release Notes: Version 4.1 (THE UNIFIED WEB BACKEND)

## Overview
Calc-Avg has achieved architectural perfection. **Version 4.1** bridges the gap between the high-performance C Engine and the stunning Standalone Web Dashboard introduced in v4.0. The C backend has been upgraded into a comprehensive API Microservice, and the web frontend has been optimized into a Thin Client that asynchronously requests complex mathematical computations directly from the C server!

## Key Features Implemented

### 1. Advanced C Backend REST API (`src/server.c`)
- **New API Route (`/api/predict`):** The C Engine natively intercepts incoming historical datasets, runs the Linear Regression Machine Learning Algorithm (calculating exact Slopes and Y-Intercepts), and returns predictive JSON payloads containing exact future data forecasts.
- **New API Route (`/api/sma`):** The C Engine natively calculates Quantitative Financial Moving Averages based on user-defined window periods and returns the computed sequences.
- **Unified File Serving:** The TCP Socket server now directly hosts the massive multi-tab `web-app/` directory (serving `index.html`, `style.css`, and `script.js` directly to the browser on Port 8080).

### 2. Frontend Thin-Client Integration (`web-app/script.js`)
- **Asynchronous Data Fetching:** The JavaScript math engines have been replaced with ultra-fast asynchronous `fetch()` API calls. 
- **C-Powered Visualizations:** When a user clicks "Train AI" or "Calculate SMA" on the web dashboard, the data is instantly routed to the C `.exe` backend, processed in pure C, and the resulting JSON is routed back to the browser where it is seamlessly rendered onto the `Chart.js` canvases!
