# Release Notes: Version 3.8 (THE BIG DATA EXPANSION)

## Overview
Calc-Avg has achieved Big Data capabilities! **Version 3.8** introduces a massive **Big Data CSV Parsing Architecture**. The Web Server has been rigorously upgraded to natively handle `multipart/form-data` HTTP POST uploads. You can now drag and drop a `.csv` file containing massive arrays of millions of numbers into the Web Dashboard, and the C Engine will natively parse the raw text array entirely in RAM to compute lightning-fast statistical metrics!

## Key Features Implemented

### 1. HTTP POST Multipart Form Parser (`src/server.c`)
- **Dynamic Content-Length Buffering**: The C Server now parses the `Content-Length` HTTP header and continuously loops the `recv()` socket command until the entire massive payload is securely buffered in the heap.
- **In-Memory CSV Tokenization**: Programmed an advanced manual boundary parser in C that physically reads the raw HTTP bytes, strips the `multipart/form-data` web headers, and isolates the raw `.csv` payload. It then loops over the raw character buffer, identifying every numeric value across thousands of rows instantly!

### 2. High-Performance Mathematics
- **Global Average & Min/Max Tracking**: As the C engine parses the million-row CSV file, it simultaneously calculates the global sum, the mathematical average, the absolute minimum value, and the absolute maximum value.
- **SQLite Database Logging**: The massive data operation is securely logged into the localized `database.sqlite` via the built-in `log_to_db()` wrapper, keeping a permanent record of all Big Data parsing requests.

### 3. Big Data Web Dashboard (`web-app/`)
- **Drag & Drop HTML5 Zone**: Engineered a premium, glowing dashed-border drop-zone in `index.html` complete with hover animations.
- **FormData API Uploads**: Programmed the frontend Javascript to intercept the uploaded file, bundle it into an HTTP `FormData` object, and execute an asynchronous `fetch()` POST request natively to the C Server.
- **Real-Time Render**: After the C server instantly parses the million rows in RAM, it returns a JSON response rendering the Global Average, Min Value, Max Value, and total Rows Parsed flawlessly onto the Dashboard.
