# Release Notes: Version 1.1

## Overview
Version 1.1 drastically expanded the mathematical capabilities of the engine, moving beyond simple dynamic arrays and introducing hard-coded mathematical sequence generators and basic file reading.

## Key Features Implemented
- **O(1) Sequence Formulas:** Implemented mathematical proofs to instantly calculate the average of Natural, Odd, and Even sequences without needing `for` loops.
- **Prime Number Optimization:** Built the `is_prime()` function using the highly optimized `O(sqrt(N))` algorithm, bypassing multiples of 2 and 3 instantly.
- **Fibonacci Integration:** Added the ability to calculate the average of the Fibonacci sequence, utilizing `long long` integers to delay integer overflow on massive sequences.
- **File Reader v1:** Implemented `calc_avg_from_file()` allowing users to point the calculator at a single `.txt` file to extract numbers.
