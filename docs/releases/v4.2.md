# Release Notes: Version 4.2 (THE ULTIMATE WEB EXPANSION)

## Overview
Calc-Avg has unleashed the absolute raw power of the C Engine onto the Web Interface! **Version 4.2** introduces the Ultimate Web Expansion. The web frontend now taps directly into the C Backend's deepest capabilities, allowing users to calculate Engineering Calculus, generate Cybersecurity Hashes, and perform Big Data Compression dynamically from the browser!

## Key Features Implemented

### 1. New C Backend APIs (`src/server.c`)
- **`/api/calculus`**: The C Engine now intercepts arrays and dynamically computes Numerical Integrals (calculating Area Under the Curve) and instant Derivatives (Rate of Change) natively over the network!
- **`/api/crypto`**: The Web Interface can now request data integrity. The C engine physically hashes data streams using a pure C implementation of SHA-256 and returns a 64-character unbreakable hexadecimal signature.
- **`/api/compress`**: The C engine runs the Run-Length Encoding (RLE) algorithm on massive data streams, instantly returning the compressed byte array and exact Compression Ratio to the browser.

### 2. Expanded Web Dashboard (`web-app/`)
- **3 Brand New Tabs**: The Glassmorphism UI now proudly displays tabs for **Calculus**, **Crypto**, and **Compression**.
- **`Chart.js` Calculus Graphing**: When integrating data, the frontend will dynamically render the area under the curve and plot the exact line of the Derivative using `Chart.js`!
