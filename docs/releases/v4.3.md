# Release Notes: Version 4.3 (THE LIVE STREAMING EXPANSION)

## Overview
Calc-Avg has officially broken the boundaries of standard HTTP! **Version 4.3** introduces **Native C WebSockets**. The C Backend Engine can now maintain a persistent, open, real-time connection directly with the Web Browser, enabling live data streaming at 20 frames per second without any browser refresh!

## Key Features Implemented

### 1. Native C WebSocket Server (`src/server.c`)
- **WebSocket Handshake Protocol**: The C engine natively intercepts the `Upgrade: websocket` HTTP header. Utilizing the Windows Cryptography API (`<wincrypt.h>`), the engine physically generates the `Sec-WebSocket-Accept` key by hashing the GUID with **SHA-1** and encoding it to **Base64** natively in C!
- **Data Streaming Thread (`websocket_stream_thread`)**: Once connected, the C engine spawns a dedicated CPU thread. This thread generates native WebSocket Data Frames (Opcode `0x81`) completely from scratch. 
- **Real-Time Data Generation**: The server dynamically calculates a continuously shifting Sine Wave blended with randomized market noise, streaming JSON X/Y coordinates directly through the TCP Socket at a massive rate of 20Hz (every 50 milliseconds)!

### 2. Live Stream Web Dashboard (`web-app/`)
- **🔴 Live Stream Tab**: Added a brand new 7th tab to the Glassmorphism Web App.
- **WebSocket Client (`new WebSocket()`)**: The Javascript seamlessly connects to `ws://localhost:8080`. 
- **High-Speed Chart.js Rendering**: When the JSON stream floods in from the C engine, `Chart.js` intercepts it, dynamically pushing the new X/Y coordinates into the dataset and shifting old data out, creating an ultra-smooth scrolling "Task Manager" or "Stock Market" style live graph!
