# Release Notes: Version 1.7

## Overview
Version 1.7 is the massive "Power User" update. It completely untethered the application from the interactive menu, allowing standard shell users to execute calculations instantly.

## Key Features Implemented
- **Command-Line Interface (CLI):** Restructured `main.c` to deeply parse `argc` and `argv`. 
- **CLI Arrays:** Users can now execute the binary with arguments (e.g. `calc_avg.exe 10 20 30`). The program detects the arguments, calculates the array instantly, prints the colored output, silently logs it to the history, and exits without ever rendering the UI.
- **CLI Files:** Added support for the `--file` flag (e.g. `calc_avg.exe --file data.txt`).
