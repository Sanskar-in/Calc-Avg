# Release Notes: Version 4.4 (LIVE SYSTEM MONITOR EXPANSION)

## Overview
Calc-Avg has achieved hardware-level intelligence! **Version 4.4** introduces the **Live Windows System Monitor**. Leveraging the WebSocket engine built in Version 4.3, the C engine now interacts directly with the Windows Kernel to extract actual CPU and RAM utilization metrics, streaming them live to the Web Dashboard like a browser-based Task Manager!

## Key Features Implemented

### 1. Windows Kernel Hardware APIs (`src/server.c`)
- **CPU Delta Calculation**: The C engine utilizes the `GetSystemTimes()` kernel API to read the raw `FILETIME` of the system's Idle, Kernel, and User threads. By precisely calculating the delta over a 100ms interval, it accurately computes the exact live CPU usage percentage!
- **RAM Utilization Tracking**: Utilizing the `GlobalMemoryStatusEx()` kernel API, the engine physically scans the computer's motherboard to retrieve the total installed physical memory and the currently available memory, mathematically computing the RAM usage percentage.

### 2. Dual-Line WebSocket Graph (`web-app/`)
- **🔴 Live Stream Tab Upgrade**: The "Live Stream" tab in the Glassmorphism Web App has been redesigned into a real-time Task Manager.
- **Dual-Axis Chart.js Integration**: The high-speed WebSocket `ws.onmessage` handler now intercepts JSON packets containing both `{"cpu": %, "ram": %}`. It dynamically pushes these two values into a beautifully color-coded Dual-Line `Chart.js` graph (Red for CPU, Blue for RAM).
- **Smooth 10Hz Broadcast**: The C engine broadcasts these system statistics completely seamlessly 10 times per second without any browser refresh or freezing.
