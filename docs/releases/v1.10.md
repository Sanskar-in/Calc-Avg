# Release Notes: Version 1.10 (LATEST)

## Overview
Version 1.10 is the ultimate culmination of the Calc-Avg project. It introduces massive data aggregation capabilities, solidifying the tool as a professional data science utility.

## Key Features Implemented
- **Multi-File Batch Processing:** Built the `calc_avg_multi_file()` engine.
- **Functionality:** Users can now pass an infinite array of `.txt` files to the program simultaneously. The engine opens each file, extracts the numbers, aggregates the massive data pool in memory, and returns the unified global average of the entire batch.
- **Integration:** This feature is fully supported in the interactive menu (Option 18) and the CLI engine using the new `--batch` flag.
