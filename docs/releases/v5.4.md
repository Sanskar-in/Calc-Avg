# Release Notes - Version 5.4.0.0 (The Terminal Aesthetics Update)

## Overview
Calc-Avg was originally born as a lightweight C Command-Line Interface (CLI) application. While it has rapidly evolved into a full-scale Web Server, Neural Network, and Advanced Analytics Engine, Version 5.4 serves as a loving homage to its console roots. We've introduced a stunning visual Easter Egg directly into the terminal interface to give it the ultimate "Masterpiece" hacker aesthetic.

## Key Features & Changes
- **Matrix Digital Rain Visualizer:** Added a highly-optimized, multi-threaded C rendering loop (`start_matrix_rain()`) that drops randomized, colorized ASCII characters vertically across the terminal.
- **Cross-Platform Console Hooks:** The visualizer dynamically hooks into the Windows `CONSOLE_SCREEN_BUFFER_INFO` API for Win32 and the `sys/ioctl.h` terminal structs for Linux/macOS to seamlessly calculate screen dimensions on any OS.
- **Menu Integration:** Accessible via a new secret menu option **(99)** on the main CLI screen.

*Calc-Avg's CLI now officially matches the gorgeous aesthetic of the Web Dashboard! Signed off by Sanskar - 2026*
