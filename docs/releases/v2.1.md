# Release Notes: Version 2.1 (THE REPORT UPDATE)

## Overview
Version 2.1 introduces autonomous Web Report generation. Moving beyond the terminal, Calc-Avg can now generate beautifully styled HTML/CSS webpages containing deep statistical analytics of your datasets!

## Key Features Implemented
- **HTML Report Generation Engine:** Built the `generate_html_report()` algorithm inside `utils.c`. 
- **Autonomous Analytics:** When triggered, this engine does not just calculate the mean. It autonomously invokes `calc_median`, `calc_mode`, `calc_variance`, and `calc_standard_deviation` simultaneously to build a complete statistical profile of the dataset.
- **Dynamic Web Rendering:** The C engine uses `fprintf` to write a complete HTML5 document. It injects a modern CSS stylesheet (`<style>`) giving the webpage premium fonts, drop shadows, and Cyan/Blue data tables. It dynamically prints the current time and date of the calculation using `<time.h>` and injects the statistical values perfectly formatted into the HTML table.
- **Integration:** This feature is accessible through the interactive menu (Option 21) or instantly via the CLI using the new `--report` flag (e.g. `calc_avg --report 10 20 30 40 50`).
