# Release Notes: Version 3.5 (THE DATABASE EXPANSION)

## Overview
Calc-Avg now possesses persistent memory! **Version 3.5** integrates the complete SQLite3 C amalgamation source code directly into the native Windows executable. Every calculation is now permanently logged into a local `database.sqlite` file, forming a complete historical record of all analytical operations!

## Key Features Implemented

### 1. SQLite Engine Integration (`sqlite3.c`)
- **Native C Compilation**: The massive 8MB `sqlite3.c` amalgamation file is now statically linked and compiled directly into `calc_avg.exe`. No external `.dll` files are required!
- **Database Initialization**: When the Web Server boots (`launch_web_server`), the engine automatically provisions the `database.sqlite` file and constructs the relational `history` SQL table (containing ID, Timestamp, Operation, Input, and JSON Result data).

### 2. Comprehensive Analytics Logging
- **`log_to_db()` Architecture**: A secure SQLite wrapper function was engineered and injected into every major API endpoint within `server.c`.
- **Permanent Logging**: The C engine now strictly executes `INSERT INTO history` for:
  - Statistics (`/api/calculate`)
  - AI Linear Regression (`/api/predict`)
  - Financial SMA (`/api/sma`)
  - Engineering Calculus (`/api/calculus`)
  - Cryptography SHA-256 (`/api/crypto`)
  - RLE Compression (`/api/compress`)

### 3. Web API History Endpoint (`/api/history`)
- **REST JSON API**: A brand new HTTP route was constructed inside `server.c`.
- **SQL Querying**: The endpoint securely executes `SELECT * FROM history ORDER BY id DESC LIMIT 50` against the SQLite database.
- **Dynamic JSON Construction**: The C thread dynamically loops over the returned SQL rows and safely constructs a massive, perfectly formatted JSON array, streaming it back to the browser.

### 4. History Log Web Dashboard Tab (`web-app/`)
- **UI Expansion**: A beautifully designed "📜 History Log" tab was added to the Glassmorphism HTML navigation bar.
- **Asynchronous Fetching**: `script.js` was upgraded with an asynchronous `fetchHistory()` Javascript function that triggers whenever the tab is clicked, parsing the C Engine's JSON response and dynamically generating HTML table rows (`<tr>`).
- **Glassmorphism CSS**: Engineered modern CSS styling for the `.history-table`, complete with glowing hover effects, translucent backgrounds, and colored `.operation-badge` elements to maintain the dashboard's premium aesthetic!
