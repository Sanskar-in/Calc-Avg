# Release Notes - Version 4.9.0.0 (The POSIX Cross-Platform Expansion)

## Overview
Version 4.9 introduces a monumental architectural shift for the Calc-Avg engine: true cross-platform compatibility. Prior versions were heavily bound to the Windows API for memory management, threads, sockets, and hardware interactions. With V4.9, the core system can now be natively compiled and executed on Linux and macOS utilizing standard POSIX libraries!

## Key Features & Changes
- **POSIX Sockets Integration:** Migrated the entire web server architecture in `server.c` to support `#ifdef _WIN32` blocks. Standard POSIX sockets (`<sys/socket.h>`, `<arpa/inet.h>`, `<unistd.h>`) dynamically replace `winsock2.h` on UNIX systems.
- **POSIX Threading (Pthreads):** The native Windows `_beginthreadex` threading mechanism has been completely abstracted. If compiling on Linux/macOS, the system seamlessly links against `<pthread.h>` to spawn worker threads and handle concurrent API requests.
- **Mutex Synchronization:** Windows `CRITICAL_SECTION` locks have been mirrored with `pthread_mutex_t` implementations to guarantee memory safety during concurrent Session ID generation and Neural Network matrix operations on UNIX systems.
- **OS-Aware Build System:** The `Makefile` has been entirely rewritten to automatically detect the host operating system (`ifeq ($(OS),Windows_NT)`). It intelligently allocates compilation flags (`-lws2_32` vs `-pthread`) based on the environment.
- **Graceful Stubbing:** Deep OS features that cannot be ported to Linux (like native GDI Desktop Capture, low-level Windows keyboard hooking, and motherboard hardware `Beep()` sonification) are gracefully stubbed out in POSIX mode to prevent compilation errors while retaining core mathematical engine capabilities.

## Security & Maintenance
- Refactored `server.c` WebSocket Handshake logic to isolate `<wincrypt.h>` and `CryptBinaryToStringA` dependencies exclusively for Windows builds.
- Ensured continuous stability of V4.8 Neural Network dynamic memory optimizations across all platforms.
