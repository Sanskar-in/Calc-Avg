# Release Notes: Version 1.8

## Overview
Version 1.8 is a major Quality of Life (QoL) update aimed at keeping the user fully immersed inside the application terminal without needing secondary windows.

## Key Features Implemented
- **In-App History Viewer:** Built the `view_history()` function.
- **Functionality:** Users can now select Option 17 in the main menu to instantly open the `history.log` file in Read Mode. The application beautifully streams the entire history (with timestamps) directly to the console wrapped in Cyan borders. 
- **Error Handling:** If the user attempts to view the history before making any calculations, the engine detects the missing file and gracefully warns them in Red without crashing the application.

## Binary Archive
The compiled Windows executable for this specific version is safely stored in:
`versions/1.8/calc_avg.exe`
