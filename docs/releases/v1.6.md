# Release Notes: Version 1.6

## Overview
Version 1.6 finalizes the core statistical engine of the application by introducing complex data spread modeling algorithms.

## Key Features Implemented
- **Variance Analysis:** Implemented `calc_variance()` to analyze the spread of a dataset around the mean.
- **Standard Deviation Analysis:** Implemented `calc_standard_deviation()` leveraging `<math.h>`'s `sqrt` function to return the standard deviation of datasets.
- **True Statistical Scoping:** Inside the interactive menu, when a user selects Variance or Standard Deviation, the UI dynamically pauses and asks the user: "Is this data a Population (1) or a Sample (2)?". The math engine then dynamically adjusts its division (`N` vs `N-1`) to calculate perfectly accurate true statistics.
