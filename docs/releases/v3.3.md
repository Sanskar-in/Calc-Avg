# Release Notes: Version 3.3 (THE REMOTE DESKTOP EXPANSION)

## Overview
Calc-Avg has officially evolved into a full-scale surveillance and streaming engine! **Version 3.3** introduces **Live Remote Desktop Screen Streaming (RDP)**. Using low-level Windows Graphics Device Interface (GDI) APIs and an upgraded 64-bit WebSocket architecture, the C engine literally captures physical screenshots of your desktop monitor and streams the live video feed directly to your Web Dashboard!

## Key Features Implemented

### 1. Windows GDI Screen Capture (`src/server.c`)
- **Native Desktop Rendering**: The C engine utilizes `GetDC(NULL)` to access the physical display monitor.
- **In-Memory Bitmaps**: Using `StretchBlt()`, the engine physically screenshots the desktop and scales it down to a high-performance `640x360` 24-bit RGB `.bmp` image natively in RAM.
- **Wincrypt Base64 Encoding**: The engine uses `CryptBinaryToStringA` from `<wincrypt.h>` to encode the massive hundreds-of-kilobytes binary blob into a single Base64 image payload in microseconds!

### 2. Upgraded 64-Bit WebSocket Architecture
- **Extended Framer (Opcode 127)**: The original WebSocket thread could only handle a maximum of 125 bytes. We have successfully upgraded the `server.c` WebSocket architecture to calculate and inject the massive 64-bit Payload Length header, allowing the engine to effortlessly blast JSON strings exceeding a megabyte perfectly across the TCP socket!
- **Data Multiplexing**: The engine simultaneously tracks CPU Usage%, RAM Usage%, and the Base64 Screen Blob, multiplexing all of them into a single massive JSON string.

### 3. Remote Desktop Web Viewer (`web-app/`)
- **Live Stream RDP Tab**: The web dashboard's Javascript intercepts the massive JSON payload, parsing the `data.screen` attribute.
- **Native Browser Painting**: The web app uses standard HTML `<img src="">` properties to rapidly paint the base64 `.bmp` image, turning the Web Dashboard into a fully functional live remote desktop!
