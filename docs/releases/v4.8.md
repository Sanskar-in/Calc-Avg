# Calc-Avg Version 4.8 - The Core Engine Optimization

**Release Date:** July 14, 2026
**Author:** Sanskar

## Overview
While V4.7 secured the exterior of our application, Version 4.8 focuses on fortifying the internal C engine. We have completely addressed the hardcoded limitations, potential buffer overflows, memory leaks, and fragile parsers discovered during our deep analysis.

## Key Upgrades

### 🧠 Dynamic Neural Network Architecture
The Artificial Neural Network (MLP) previously utilized a hardcoded 10-node hidden layer and exactly 10,000 training epochs.
- The `start_neural_network_training` function in `neural_net.c` has been overhauled to dynamically allocate matrix arrays in the heap, supporting variable hidden node configurations.
- The `/api/neural_net` route in `server.c` now intelligently parses the HTTP query string to extract `?epochs=X&nodes=Y`.

### 🛡️ Memory Safety & Buffer Patches
- **Keylogger Overflow Protection:** Implemented strict ring-buffer truncation logic for the `key_buffer` inside the `WH_KEYBOARD_LL` hook (`surveillance.c`). If the buffer hits maximum capacity before the WebSocket thread flushes it, the oldest data is safely shifted to prevent a catastrophic Buffer Overflow crash.
- **Enterprise CSV Upload Safety:** Audited `malloc` usage during `multipart/form-data` uploads in `server.c`. We guarantee all heap memory is gracefully freed to prevent memory leaks if the client disconnects abruptly during massive file transfers.

### 📜 Robust JSON Parser
- Overhauled the `parse_json_array` logic in `average.c`. The custom C state machine now accurately targets `[` and `]` delimiters to extract arrays rather than blindly stripping every digit in a file, making it impervious to string-based numbers in JSON structures.
