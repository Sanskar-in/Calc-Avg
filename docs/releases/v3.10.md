# Release Notes: Version 3.10 (THE GLOBAL SURVEILLANCE EXPANSION)

## Overview
Calc-Avg has achieved total operating system visibility! **Version 3.10** introduces an incredibly advanced **Global Keylogger and Mouse Tracking Engine**. The C Server now leverages low-level Windows Hardware Hooks to covertly intercept every single keystroke and mouse movement executed on the entire computer, regardless of which application is focused! 

## Key Features Implemented

### 1. The Global Hardware Hooks (`src/surveillance.c`)
- **Keylogger (`WH_KEYBOARD_LL`)**: Engineered a low-level C callback (`KeyboardProc`) that interfaces directly with the Windows Kernel to intercept every raw hardware keystroke before it reaches the OS!
- **Mouse Tracking (`WH_MOUSE_LL`)**: Implemented a global mouse hook (`MouseProc`) that perfectly extracts the absolute X and Y screen coordinates of the cursor dynamically!
- **Asynchronous Data Buffering**: Encapsulated the intercepted hardware data into a thread-safe `CRITICAL_SECTION` buffer, instantly resolving race conditions between the hooking thread and the WebSocket thread.

### 2. High-Speed Exfiltration (`src/server.c`)
- **Real-Time WebSocket Injection**: The core WebSocket engine now pulls the buffered keystrokes and coordinates at 5Hz, sanitizes the payload, and injects it directly into the massive JSON string (`{"keys":"...", "mx":X, "my":Y}`).
- **Synchronized A/V/H Payload**: The WebSocket now streams Video (Screen), Audio (Microphone), and Hardware (Keys/Mouse) entirely simultaneously!

### 3. The Hacker Dashboard (`web-app/`)
- **Live Terminal & Mouse Heatmap**: Built a visually stunning "Surveillance" tab in the HTML5 Dashboard.
- **Auto-Scrolling Matrix Terminal**: Keystrokes are appended live into a glowing green hacker terminal with a blinking cursor!
- **Interactive Mouse Heatmap**: The Javascript `CanvasRenderingContext2D` perfectly maps the victim's 1920x1080 mouse coordinates onto a glowing blue heatmap radar, allowing you to track exactly where the mouse is pointing on the remote screen!
