# User Suggestions & Contributing

Calc-Avg is an open-source project by **Sanskar**! We heavily rely on the community to suggest next steps, request features, and contribute code.

## How to Suggest Next Features

If you are a user and have a brilliant idea for the next feature (e.g., adding GUI support, supporting JSON files, or adding calculus functions):

1. **Visit the GitHub Repository:** Navigate to [https://www.github.com/sanskarIN/Calc-Avg](https://www.github.com/sanskarIN/Calc-Avg).
2. **Open an Issue:** Click on the "Issues" tab.
3. **Submit a Feature Request:** Click "New Issue", give it a clear title (e.g., "Feature Request: JSON Support"), and thoroughly explain what you want the feature to do.

The maintainer (Sanskar) or other open-source contributors will review your suggestion and implement it in a future version!

## How to Contribute Code

If you are a developer and want to build the next feature yourself:

1. **Fork the Repository:** Click the "Fork" button on the GitHub page to create your own copy of the codebase.
2. **Clone your Fork:** `git clone https://www.github.com/YOUR_USERNAME/Calc-Avg`
3. **Create a Branch:** `git checkout -b feature/MyAwesomeFeature`
4. **Code your Feature:** Add your feature to the `src/` files. Ensure you update `README.md` and the `docs/`!
5. **Commit & Push:** `git commit -m "Added MyAwesomeFeature"` and `git push origin feature/MyAwesomeFeature`.
6. **Open a Pull Request:** Go back to the original Calc-Avg repository and click "New Pull Request".

We are actively seeking ideas to push Calc-Avg to Version 2.0 and beyond!
