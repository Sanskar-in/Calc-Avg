# Version 4.0: The Global Web App Release

You will notice there is no `calc_avg.exe` file inside this `4.0` directory!

This is because **Version 4.0** was purely a Frontend Architectural Milestone. During Version 4.0, we engineered a completely independent, 100% Client-Side Web Application stored in the root `web-app/` folder.

In this iteration, the `web-app/script.js` file calculated all the advanced statistics, predictive AI, and financial Moving Averages entirely natively in the browser without requiring any C `.exe` backend to run. 

This made Version 4.0 the first "Standalone Global Release" capable of being hosted statically on GitHub Pages.

In **Version 4.1**, we bridged the two worlds back together: we unified this beautiful standalone Web Dashboard with the high-performance C Engine API, returning the `calc_avg.exe` to the `versions/4.1/` folder as the master brain of the operation!
